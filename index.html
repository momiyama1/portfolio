掌紋認証システム

このプロジェクトは、Webブラウザ上で動作する掌紋認証システムです。 ユーザーの手のひら（掌紋）をカメラで撮影し、その特徴点を抽出して認証を行います。 AKAZEアルゴリズムを用いた特徴点抽出とBrute Force Matcherによるマッチング技術を組み合わせ、 高精度な生体認証を実現しています。

Python Flask OpenCV AKAZEアルゴリズム HTML5 Canvas JavaScript (Web API)

実装した主な機能:

  • ブラウザ上でのリアルタイムカメラ映像表示と撮影範囲指定
  • 画像の前処理（グレースケール変換、コントラスト調整、照明補正、ノイズ除去）
  • AKAZEアルゴリズムによる特徴点抽出と可視化
  • Brute Force MatcherとRANSACを用いた高精度なマッチング処理
  • 認証結果と特徴点マッチングの視覚的フィードバック

このシステムは、生体認証技術の実用的な応用例として開発しました。
従来のパスワード認証に代わる、より安全で使いやすい認証方式を目指しています。
Webアプリケーションとして実装することで、特別なソフトウェアのインストールなしで 誰でも簡単に試すことができます。

カレンダーアプリ

このプロジェクトは、C#とWPFを使用して作成したデスクトップ用のカレンダーアプリです。 月ごとのカレンダーを表示し、日付ごとに予定を追加、編集、削除できる機能を提供します。 予定はJSON形式で保存され、簡単に管理できるようになっています。

C# WPF JSON Local Storage

実装した主な機能:

  • 月ごとのカレンダー表示
  • 日付ごとの予定追加、編集、削除
  • 予定の入力・編集用ダイアログ
  • 予定をJSON形式で保存
  • 直感的なUIと操作感

このカレンダーアプリは、個人の予定管理をシンプルかつ効果的に行うために作成しました。
予定の入力・編集も簡単で、日々の生活に役立つツールです。

ToDoアプリ

このプロジェクトは、Reactを使って作成したToDoリストアプリです。 ユーザーがタスクを追加、編集、削除できる機能を持ち、期限を設定することも可能です。 また、期限が過ぎたタスクは赤く表示され、進捗状況に応じて完了したタスクの表示が変更されます。

React JavaScript CSS JSON Local Storage

実装した主な機能:

  • タスクの追加、編集、削除
  • 期限設定機能
  • 期限が過ぎたタスクを赤色で表示
  • タスク完了状態をチェックボックスで切り替え
  • タスクのデータをローカルストレージに保存

このアプリは、タスク管理を効率的に行いたいユーザーに向けて開発しました。
Reactの状態管理とローカルストレージを活用して、シンプルで使いやすいUIを提供しています。

太陽系シミュレーション

このプロジェクトは、Unityを使用して作成した太陽系シミュレーションです。 太陽系の惑星たちを3Dで表示し、実際の軌道に近い動きを再現しています。

Unity C# 3Dモデル

実装した主な機能:

  • 太陽系の主要な惑星（水星、金星、地球、火星、木星、土星、天王星、海王星）の3D表示
  • 実際の軌道に近い惑星の公転運動
  • 惑星の自転
  • カメラコントロールによる視点変更
  • 惑星の情報表示

このシミュレーションは、太陽系の惑星をリアルに再現し、視覚的に楽しむことができるゲームです。
ユーザーは惑星の軌道を観察しながら、自由にカメラを動かすことができます。
